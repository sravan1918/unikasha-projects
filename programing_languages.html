List of Programming Languages

Name Year of Creation Description
Python 1991 A high-level, general-purpose programming language known for its readability and simplicity.
JavaScript 1995 A popular scripting language primarily used for web development, enabling interactive and dynamic web content.
Java 1995 A versatile, object-oriented language widely used for building enterprise-scale applications, Android apps, and more.
C 1972 A low-level language commonly used for system programming and embedded systems.
C++ 1983 An extension of the C language with added features, used for building high-performance applications and games.
C# 2000 A modern, object-oriented language developed by Microsoft, primarily used for building Windows applications and .NET development.
Ruby 1995 A dynamic, reflective language known for its simplicity and productivity, often used for web development and scripting.
Swift 2014 An intuitive, open-source language developed by Apple for iOS, macOS, watchOS, and tvOS app development.
